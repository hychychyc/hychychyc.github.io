最大正方形

dp
处理一个矩阵前缀和，就可以O(n)的求出矩阵的和了，而且只有和为完全平方数才是正方形，因为数据只有1吗，要是不是一的话就不容易了
这样再枚举一个起点，和一个边长就是任意一个正方形了，再优化剪枝一下，就可以0ms过了

真正的dp，之前前缀和也算是dp吧
就是动态转移方程f[i][j]=a[i][j]*(min(f[i-1][j],min(f[i][j-1],f[i-1][j-1]))+1);
也就是说又可以组成正方形的最小前驱扩展来的。
